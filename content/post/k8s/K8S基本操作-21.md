我們可以採用以下方式設置環境變數:

```yaml
apiVersion: v1
kind: Pod
metadata:
  labels:
    name: webapp-color
  name: webapp-color
  namespace: default
spec:
  containers:
  - env:
    - name: APP_COLOR
      value: pink
    image: kodekloud/webapp-color
    imagePullPolicy: Always
    name: webapp-color
```

其中的這一段就是了:

```yaml
  - env:
    - name: APP_COLOR
      value: pink
```
