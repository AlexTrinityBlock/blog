# 命令建立 Secret

假定我們的 Pod 稱為 app 。

例如，以下的的指令用來直接看 Pod 中的 Log (當然，這不是好方法，但在一切裝置好之前，也許會用上):

```bash
kubectl exec -it app cat /log/app.log
```

或者用以下指令連入 Pod 的 Shell:

```bash
kubectl exec -it app sh
```
