# Secure Context 設置容器為 no-root 用戶

我們可以加入以下這段:
```yaml
spec:
  securityContext:
    runAsUser: 1010
```

加完之後狀況如下:

```yaml
---
apiVersion: v1
kind: Pod
metadata:
  name: ubuntu-sleeper
  namespace: default
spec:
  securityContext:
    runAsUser: 1010
  containers:
  - command:
    - sleep
    - "4800"
    image: ubuntu
    name: ubuntu-sleeper
```

這樣就會以非 Root 的 User ID 1010 執行容器了。
