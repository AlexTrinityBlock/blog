**第一題**

**第二題**

網段 Mask 計算，遮罩 255.255.255.192 可用的 IP 範圍。

**第三題**

如何監看某個 port 被哪個 process 占用。

**第四題**

TCP/UDP的差異?

**第五題**

如何設置 Nginx 的設置讓一個 Server IP 擁有兩個 Domain Name。需要強記 Nginx 的 Config 檔。

**第六題**

然後撰寫一個 deployment.yaml 來對 K8S 某個 Pod 產生的 Log 與 stdout 進行合理監測。
