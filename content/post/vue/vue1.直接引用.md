# Vue筆記 1：直接引用

## 該筆記為了快速上手，蓄意忽視webpack等套件使用，若有機會將會於別篇筆記提及

首先我們簡單的建立一個html檔案，然後輸入以下內容

```html
<!DOCTYPE html>
<html lang="zh-hant-TW">

<head>
    <meta charset="UTF-8">
    <title>簡化版Vue學習</title>
</head>

<body>

</body>
<!--此行引入vue.js-->
<script src="https://cdn.jsdelivr.net/npm/vue"></script>

</html>
```
