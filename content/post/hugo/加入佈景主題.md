# 從github抓取的指令

例如 ，我們要抓的佈景主題的名稱叫 m10c   
我們找到m10c的部落格網址   
然後輸入以下指令   

```
git submodule add https://github.com/vaga/hugo-theme-m10c.git themes/m10c
```

指令的最後一段    
``` 
themes/m10c 
```      
意思是將主題模塊放入 
```
/blog/themes/m10c
```
