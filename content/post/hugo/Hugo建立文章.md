### 1.命令行

命令行輸入後  

```
hugo new posts/心平氣和.md
```

這樣會建立一個稱為『心平氣和』的文章
  
### 2.編輯

可以用各種文字編輯器進行編輯   
通常是輸入內容
  
### 3.伺服器預覽文章

命令行輸入   

```
hugo server -D
```   

可以進行預覽

### 4.生成檔案

正式建立靜態文章  

```
hugo server -D
```   

就會生成文章的html檔案，在public資料夾了
