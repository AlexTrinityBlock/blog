最近 Hugo

加入佈景主題

從github抓取的指令 例如 ，我們要抓的佈景主題的名稱叫 m10c 我們找到m10c的部落格網址 然後輸入以下指令 git submodule add https://github.com/vaga/hugo-theme-m10c.git themes/m10c 指令的最後一段 themes/m10c 意思是將主題模塊放入 /blog/themes/m10c 資料夾

建立一個新的部落格

1.命令行 命令行輸入後 hugo new site hahablog 這樣會建立一個稱為『haha』的部落格
繼續閱讀

Hugo建立文章

1.命令行 命令行輸入後 hugo new posts/心平氣和.md 這樣會建立一個稱為『心平氣和』的文章 2.編輯 可以用各種文字編輯器進行編輯 通常是輸入內容 3.伺服器預覽文章 命令行輸入 hugo server -D 可以進行預覽 4.生成檔案 正式建立靜態文章 hugo server -D 就會生成文章的html檔案，在public資料夾了
